土地資料系統 簡短版變更紀錄
================================

1. 系統初始方向
- 由原本想做 SQLite 客戶資料系統，整理成土地資料管理系統。
- 使用 SQLite 作為本機資料庫。

2. 欄位與資料結構調整
- 移除 Email 欄位。
- 原本的公司欄位改為 ID 使用。
- 主要欄位整理為：
  區、段、登記次序、地號、面積、公告現值、權利範圍、總公告現值、
  所有權人、ID、地址、登記原因、備註。
- 後續再調整為：
  地區、地段、序號、地號、面積/m2、公告現值、分子、分母、坪數、
  總現值/元、姓名、身分證、地址、原因、備註、出訪記錄。

3. UI 與操作調整
- 改成桌面 UI 介面。
- 右側表單版面改回接近一開始的樣式。
- 新增右鍵選單刪除客戶/資料功能。
- 新增勾選方格，可多選清除資料。
- 新增全部清除功能，並加上 3 次確認。

4. 自動計算與格式化
- 總公告現值改為自動公式計算。
- 總公告現值取消小數點。
- 公告現值與總公告現值加上千分位。
- 坪數顯示到小數點後兩位。

5. Excel 匯入
- 新增 `.xlsx` 匯入功能。
- 改善中文欄位與合併儲存格匯入。
- 匯入時可自動判斷表頭、補欄位、帶入公式。

6. 登入與加密
- 新增登入帳號驗證。
- 單人使用版本帳號固定為 `admin`。
- 登入密碼同時作為資料解密密碼。
- 敏感資料改為加密儲存。
- 新增「加密既有資料」功能。
- 新增修改密碼功能，放在「設定」內。

7. 身分證顯示控制
- 新增顯示/隱藏身分證按鈕。
- 隱藏格式為：保留前 3 碼與最後 1 碼，中間改成 `*`。

8. 備份與還原
- 新增自動備份 `customers.db`。
- 啟動時每日自動備份。
- 匯入、批次刪除、全部清除、加密既有資料、修改密碼前自動備份。
- 設定內新增「開啟備份資料夾」。
- 設定內新增「還原備份」。
- 還原前會先自動備份目前資料。

9. 效能與 UI 升級
- 原本 Tkinter 版本在筆電大量資料滾動時有明顯頓挫。
- 後續改為 PySide6 版桌面 UI。
- 保留原功能與大致版面。
- 表格改為較順的 Qt 表格模型。
- 加入分批載入，改善大量資料刷新速度。

10. 篩選與排序
- 新增欄位篩選。
- 新增欄位排序。
- 可選擇不同欄位做查找與新到舊/舊到新排序。

11. 選取格修正
- 修正用戶回報的選取格無法正常使用問題。
- 現在點擊表格第一欄可穩定切換勾選狀態。

12. EXE 與可攜化
- 新增可攜路徑處理。
- 新增打包 EXE 腳本。
- 成功打包成可在另一台 Windows 電腦使用的 EXE。

13. 目前正式版本重點
- SQLite 本機資料庫
- PySide6 桌面 UI
- Excel 匯入
- 登入驗證
- 資料加密
- 修改密碼
- 自動備份
- 備份還原
- 欄位篩選與排序
- 可攜 EXE
